========================================
  Excel Agent  使用说明
========================================

【简介】
Excel Agent  是一个基于AI的Excel/文本文件处理工具。
支持通过自然语言描述，让AI自动生成处理技能，将txt/xls/xlsx文件转换为格式化的Excel输出。

========================================
一、首次启动
========================================

1. 双击 启动Agent.bat 启动程序,也支持再终端直接运行 python agent.py (全程在终端处理)
2. 浏览器会自动打开 http://localhost:5000
3. 首次启动会自动生成以下文件夹：
   - skill/        内置技能目录（温升曲线等）
   - ai_skills/    AI生成的技能目录
   - pending/      待处理文件目录
   - output/       处理输出目录
   - uploads/      上传文件目录

========================================
二、配置AI API（必须）
========================================

使用AI生成技能前，必须先配置API Key：

1. 打开网页后，找到顶部的"配置"区域
2. 选择"模型提供商"（如 DeepSeek、OpenAI、通义千问等）
3. 输入你的 API Key
4. 其他选项（API地址、模型名称）通常会自动填充
5. 点击"保存"，然后点击"测试连接"
6. 看到"连接成功"即可使用

支持的模型提供商：
  - DeepSeek（推荐，价格便宜）：https://api.deepseek.com/v1
  - OpenAI（GPT-4o）：https://api.openai.com/v1
  - 通义千问（Qwen）：https://dashscope.aliyuncs.com/compatible-mode/v1
  - 智谱AI（GLM-4）：https://open.bigmodel.cn/api/paas/v4
  - Moonshot（Kimi）：https://api.moonshot.cn/v1
  - Anthropic（Claude）：https://api.anthropic.com
  - 百川、MiniMax、讯飞星火等更多...

========================================
三、使用内置技能
========================================

1. 点击左侧"技能列表"中的技能（如"温升"）
2. 选择要处理的文件（支持 .txt、.xls、.xlsx）
3. 点击"开始处理"
4. 处理完成后点击"下载"获取结果

内置技能"温升"功能：
  - 读取温升测试数据（txt或Excel格式）
  - 自动识别时间列和温度数据列
  - 生成格式化的Excel文件
  - 自动绘制温度曲线折线图

========================================
四、用AI生成新技能
========================================

1. 点击左侧"创建新技能"按钮
2. 填写技能名称（如"数据筛选"）
3. 描述你的处理需求（越详细越好）
4. 可选：上传"处理前示例"和"处理后示例"供AI参考
5. 点击"生成技能"
6. 等待AI返回代码（通常10-30秒）
7. 技能创建成功后会出现在技能列表中

示例需求描述：
  - "读取txt文件，按制表符分隔，提取温度数据列，转为xlsx格式，超过80度的数据标红"
  - "读取Excel文件，筛选出温度大于60度的行，生成新的Excel文件"
  - "读取txt文件，合并多列数据，生成柱状图"

========================================
五、修改AI技能
========================================

1. 点击技能列表中的AI技能
2. 点击"修改技能"按钮
3. 输入修改说明（如"增加异常值检测"）
4. AI会在现有代码基础上进行修改

========================================
六、关于txt文件处理
========================================

本工具完全支持将txt文件处理成Excel格式：
  - 自动检测文件编码（UTF-8、GBK、GB2312等）
  - 自动识别分隔符（制表符、逗号、空格、竖线等）
  - AI生成的技能会自动处理这些格式转换

========================================
七、文件路径说明
========================================

exe所在目录下的文件夹：
  exe/
    ExcelAgent.exe      主程序（双击启动）
    使用说明.txt         本文件
    skill/              内置技能
    ai_skills/          AI生成的技能
    pending/            待处理文件
    output/             处理结果输出
    uploads/            临时上传文件

处理结果路径规则：
  输出文件名格式：原名_时间戳_after.xlsx
  例如：灰0070_20260518_163000_after.xlsx

========================================
八、常见问题
========================================

Q: 双击exe后浏览器没有自动打开？
A: 手动在浏览器中输入 http://localhost:5000 访问

Q: AI生成技能报错"未配置API Key"？
A: 请先在顶部配置区输入API Key并保存

Q: txt文件处理后乱码？
A: 确保txt文件能被正常打开（记事本查看不乱码）

Q: 处理报错"缺少依赖库"？
A: 这是打包版本自带的依赖，通常不需要额外安装

Q: 如何停止程序？
A: 关闭命令行窗口，或在任务管理器中结束ExcelAgent进程
